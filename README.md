# 11 Studio — Creative Agency Website

A premium cinematic agency website built for **11 Studio**, a creative media brand focused on helping creators, brands, and businesses grow through high-quality visual storytelling.

## Overview

11 Studio provides services in:

* Video Editing
* Content Creation
* Motion Graphics
* Digital Storytelling
* Brand Promotion

This project is a modern, responsive web experience designed to reflect the identity of a high-end creative agency.

The website combines:

* Dark cinematic visuals
* Premium UI/UX
* Smooth motion animations
* Strong conversion-focused layout

---

## Design Philosophy

The website is designed to feel like a combination of:

* A film production studio
* A luxury creative agency
* A futuristic digital experience

Core design goals:

* Premium visual identity
* Strong first impression in under 5 seconds
* Clear service communication
* High conversion for client inquiries
